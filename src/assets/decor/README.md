# Decorative Assets

Use this folder for reusable decorative SVGs, section background shapes, masks, and small visual accents.

Suggested structure:

- `backgrounds/` - large section background SVGs or patterns
- `icons/` - small decorative icons and UI-like symbols
- `shapes/` - abstract shapes, waves, stars, stickers, and masks

Keep optimized SVGs here when they are imported by components. Put large raster media in a separate image folder if needed.
